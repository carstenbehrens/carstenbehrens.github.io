## Parameter

A parameter is a **variable** that gets used in the **declaration** of a function/method.

```javascript
// num is the parameter
function double(num) {
  return num * 2;
}
```

## Argument

A argument is an actual value that gets passed to the function/method.

```javascript
// The argument is 43
double(43);
```
